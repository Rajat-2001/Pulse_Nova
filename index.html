About PulseNova

PulseNova is a personal AI assistant that automates productivity tools — Gmail, Google Calendar, document knowledge retrieval, and voice interaction — through natural language commands. It runs entirely locally using LLaMA 3.1 8B via LM Studio, ensuring full privacy.

Tech Stack
FastAPI LLaMA 3.1 8B LM Studio ChromaDB Faster-Whisper gTTS Google Calendar API Gmail IMAP Discord.py Sentence Transformers Python 3.11
Features
Natural language chat via LLaMA 3.1 8B
Voice input (Faster-Whisper STT) + audio output (TTS)
Gmail read, summarize & send with LLM composition
Google Calendar create, modify & delete events
RAG over PDFs & emails with ChromaDB
Discord bot with intent detection orchestrator
100% local — no external AI APIs
